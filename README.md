# UTN DevOps - Práctica 1

Proyecto realizado para la práctica exploratoria de la materia DevOps UTN.

## Descripción

Este proyecto utiliza Vagrant y VirtualBox para crear una máquina virtual Ubuntu con un servidor web configurado automáticamente mediante provisioning.

## Tecnologías utilizadas

- Vagrant
- VirtualBox
- Ubuntu
- Apache/Nginx
- Git & GitHub

## Requisitos

- Vagrant >= 2.2.7
- VirtualBox
- Git

## Ejecución

Clonar el repositorio:

```bash
git clone https://github.com/omorera2488/utn-devops.git

Ingresar al proyecto:

cd utn-devops

Cambiar a la rama de la práctica:

git checkout unidad-1-vagrant

Levantar la máquina virtual:

vagrant up --provision
Acceso a la aplicación

Abrir en el navegador:

http://localhost:8080
